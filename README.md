# Cuadro de Mandos - Mercado de Vehículos en EE.UU.

Este proyecto es una aplicación web interactiva desarrollada en Python utilizando **Streamlit** y **Plotly**. Su objetivo es realizar un análisis exploratorio de datos (EDA) sobre un conjunto de datos de anuncios de venta de coches usados en EE.UU.

## Funcionalidades
- **Métricas Clave:** Visualización en tiempo real del total de anuncios, precio promedio y kilometraje.
- **Filtros Dinámicos:** Permite filtrar vehículos por marca, rango de precio y condición.
- **Gráficos Interactivos:** Opción de renderizar histogramas de distribución de precios y gráficos de dispersión (Precio vs Kilometraje) mediante casillas de verificación.
